# 📊 Data Dictionary: POIs (Landmarks & Restaurants)

เอกสารนี้รวบรวมคำอธิบายคอลัมน์ทั้งหมดที่ปรากฏอยู่ในไฟล์ CSV (เช่น `landmarks_clean.csv`, `restaurants.csv`) หลังจากผ่านกระบวนการทำความสะอาด (Data Cleaning) และการผสานข้อมูล (Data Enrichment) แล้ว

> [!NOTE]
> ทั้งสคริปต์ **Landmarks** และ **Restaurants** ใช้โครงสร้าง Data Dictionary นี้ร่วมกัน 100%

---

## 📍 ข้อมูลตำแหน่งที่ตั้ง (Location)

| คอลัมน์ | ชนิดข้อมูล | คำอธิบาย | ตัวอย่างข้อมูล |
| :--- | :--- | :--- | :--- |
| **province** | String | ชื่อจังหวัด (ภาษาไทย) | `"ขอนแก่น"` |
| **province_en** | String | ชื่อจังหวัด (ภาษาอังกฤษแบบมาตรฐาน) | `"Khon Kaen"` |
| **district** | String | ชื่ออำเภอ (ภาษาไทย) | `"เมืองขอนแก่น"` |
| **sub_district** | String | ชื่อตำบล (ภาษาไทย) | `"ในเมือง"` |
| **lat** | Float | ละติจูด (Latitude) จุดศูนย์กลางสถานที่ | `16.4322` |
| **lon** | Float | ลองจิจูด (Longitude) จุดศูนย์กลางสถานที่ | `102.8236` |

---

## 🏛️ ข้อมูลพื้นฐานสถานที่ (Basic Info)

| คอลัมน์ | ชนิดข้อมูล | คำอธิบาย | ตัวอย่างข้อมูล |
| :--- | :--- | :--- | :--- |
| **name** | String | ชื่อสถานที่ (ส่วนใหญ่เป็นภาษาไทย หากไม่มีจะเป็นอังกฤษ) | `"โรงพยาบาลศรีนครินทร์"` |
| **name_en** | String | ชื่อสถานที่ (ภาษาอังกฤษ - เฉพาะ OSM ที่มี tag name:en) | `"Srinagarind Hospital"` |
| **category** | String | หมวดหมู่หลักของสถานที่ | `"โรงพยาบาล"`, `"ร้านอาหาร"` |
| **layer** | Integer | ระดับความสำคัญ (1 = ใหญ่มาก/พื้นที่กว้าง, 2 = ระดับกลาง, 3 = เล็ก/ย่อย) | `1`, `3` |
| **layer_name** | String | ชื่อกลุ่มความสำคัญตาม Layer | `"สถานที่ราชการและสาธารณูปโภค"` |
| **source** | String | แหล่งที่มาดั้งเดิมของจุด POI นี้ | `"OpenStreetMap / Overpass API"`, `"Google Maps"` |

---

## 🗺️ ข้อมูลอ้างอิงจาก OpenStreetMap (OSM Data)
ข้อมูลกลุ่มนี้จะมีค่าก็ต่อเมื่อ `source` เริ่มต้นมาจาก OSM เท่านั้น (หากมาจาก Google Maps จะเป็นค่าว่างหรือ Null)

| คอลัมน์ | ชนิดข้อมูล | คำอธิบาย | ตัวอย่างข้อมูล |
| :--- | :--- | :--- | :--- |
| **osm_version** | Integer | จำนวนครั้งที่สถานที่นี้ถูกแก้ไขใน OSM (1 = เพิ่งสร้างและไม่เคยถูกแก้) | `1`, `4` |
| **osm_timestamp** | String | เวลา (Timestamp) ล่าสุดที่มีคนเข้าไปแก้ไขสถานที่นี้ในระบบ OSM | `"2024-12-15T10:30:00Z"` |
| **osm_last_edit_year_ce** | Integer | ปี ค.ศ. ล่าสุดที่มีการแก้ไขข้อมูลใน OSM | `2024` |
| **osm_last_edit_year_be** | Integer | ปี พ.ศ. ล่าสุดที่มีการแก้ไขข้อมูลใน OSM (คำนวณจาก CE + 543) | `2567` |
| **osm_created_year_ce**| Integer | ปี ค.ศ. ที่สร้างสถานที่นี้ครั้งแรก (ดึงมาได้เฉพาะตอนที่ `osm_version` = 1) | `2024` |
| **osm_created_year_be**| Integer | ปี พ.ศ. ที่สร้างสถานที่นี้ครั้งแรก | `2567` |

---

## ⭐️ ข้อมูลอ้างอิงจาก Google Maps (Enrichment Data)
ข้อมูลเหล่านี้มาจาก Google Maps ทั้งแบบดึงมาใหม่ และแบบที่ผสาน (Enrich) ให้กับร้านเดิมของ OSM

| คอลัมน์ | ชนิดข้อมูล | คำอธิบาย | ตัวอย่างข้อมูล |
| :--- | :--- | :--- | :--- |
| **gmaps_last_review_year_ce** | Integer | ปี ค.ศ. ของรีวิวล่าสุดที่ดึงมาได้จากหน้า Google Maps | `2024` |
| **gmaps_last_review_year_be** | Integer | ปี พ.ศ. ของรีวิวล่าสุดที่ดึงมาได้จากหน้า Google Maps | `2567` |
| **gmaps_status** | String | สถานะการเปิด/ปิดร้าน (จะมีค่าเมื่อร้านปิดถาวรหรือชั่วคราว) | `"Permanently Closed"`, `Null` |
| **gmaps_source_matched** | String | ป้ายกำกับแจ้งว่า POI ของ OSM ตัวนี้ ได้รับการผสานรีวิวจาก Google Maps แล้ว | `"Google Maps (Enriched)"`, `Null` |

---

> [!TIP]
> **วิธีวิเคราะห์ปีที่ร้านอาหารเปิด/อัปเดต:**
> หากต้องการหาร้านใหม่ๆ แนะนำให้ดูที่ความสดใหม่ของรีวิว (`gmaps_last_review_year_be`) เป็นหลักควบคู่ไปกับ `osm_last_edit_year_be` ครับ
